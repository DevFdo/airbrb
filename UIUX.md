# UI and UX
1. User have an avatar based on the initial of their email.

2. The owner of a property can edit, delete, publish and unpublish their property in detail page. They are also blocked to make a booking

3. The images are immediately displayed after uploaded.

4. While on load, there is a Circular Progress displayed.

5. User is able to view all the images for my listings and all published listings.

6. User returns to the previous page when delete or edit a listing since there are two paths to do so.
