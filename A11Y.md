# Assessibility

1. For icons and images, there are labels and alt to describe the usage or content of the icon or images. For icon buttons, there are also title access to explain their usage in a clear way.

2. In the home screen, all input and filters have `aria-label` that describes its function.`aria-live` is used to notify the screen reader the change while filter and sorting is applied.

3. Data displayed have `aria-label` applied, especially for the pure image data like rating.

4.  Key control is available for pages. User is able to switch button by pressing tab keys

5. Wide, horizontal pages, like home page and detail page, is implemented in a mobile responsive way that works on smaller screens.

6. For date pickers, user's can't pick date that is earlier than today.
